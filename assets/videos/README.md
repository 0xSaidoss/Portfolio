# Vidéo de fond Hero

## Comment obtenir la vidéo

Télécharge UNE de ces vidéos gratuites, renomme-la `hero-bg.mp4` et place-la dans ce dossier :

### Option 1 — Mixkit (recommandé, libre de droits)
1. Va sur https://mixkit.co/free-stock-video/programmer-hacking-a-system-on-the-laptop-46639/
2. Clique "Free Download" → choisir résolution 1080p
3. Renomme le fichier téléchargé en `hero-bg.mp4`
4. Place-le ici : `assets/videos/hero-bg.mp4`

### Option 2 — Autre vidéo Mixkit hacking
https://mixkit.co/free-stock-video/hacker-on-the-dark-web-49126/

### Option 3 — Pixabay
https://pixabay.com/videos/hacker-hacking-cyber-security-code-147701/

## Après ajout de la vidéo
```
git add assets/videos/hero-bg.mp4
git commit -m "Add hero background video"
git push
```

Le site utilisera automatiquement cette vidéo locale en priorité.
